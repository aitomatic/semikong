# Skill: Provenance Curation

## Goal

Capture evidence quality and traceability for ontology assertions.

## Procedure

1. Add provenance metadata in TTL for new/changed terms.
2. Distinguish draft/synthetic from externally verified evidence.
3. Record promotion decisions in `ontologist/01-docs/DECISIONS.md` when status changes are policy-relevant.
4. Log evidence impact in `ontologist/04-context/WORKLOG.md`.

## Done Criteria

- Provenance fields present for edited assertions.
- Any policy-impacting source decision captured in governance log.
