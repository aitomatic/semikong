# Prompt Templates

Reusable prompt templates for invoking ontologist agents with consistent scope, rules, and deliverables.

Usage:
1. Copy a template.
2. Adjust target module/path and domain-specific goals.
3. Keep control-plane references and process requirements intact.

Available templates:
1. `agent-prompt-materials-chemicals-curation.md` - focused chemicals curation run.
2. `08-materials-curation-sammy.md` - full materials module curation run.
3. `facets-dimensions-template.md` - required ontology facets/dimensions checklist block.
4. `07-wfe-06-etch-systems-curation-sammy.md` - focused etch-systems curation prompt with mandatory facets coverage.
